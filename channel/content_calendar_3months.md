## DataTable Schema (n8n `content_calendar`)
```
Fields:
- id              (auto)
- video_number    (number)   — 1 to 52
- week            (number)   — 1 to 13
- publish_date    (string)   — YYYY-MM-DD
- title           (string)   — English video title
- description     (string)   — Full English description (500–1000 chars)
- tags            (string)   — comma-separated tags
- category        (string)   — money / investment / crypto / marketing / cars / partnerships
- youtube_cat_id  (string)   — 27=Education, 22=People, 25=News, 28=Science
- video_file_url  (string)   — Google Drive / Cloudinary / S3 link
- thumbnail_url   (string)   — public image URL (1280×720)
- status          (string)   — pending / uploading / uploaded / failed
- uploaded_at     (string)   — ISO timestamp set after upload
```

## Zapier Zap Setup (Step-by-Step)
1. **Trigger:** Webhooks by Zapier → Catch Hook → copy the Hook URL
2. **Action 1:** YouTube → Upload Video
   - Title: `{{title}}`
   - Description: `{{description}}`
   - Tags: `{{tags}}`
   - Category ID: `{{youtube_cat_id}}`
   - Video File URL: `{{video_file_url}}`
   - Privacy Status: `public`
3. **Action 2:** YouTube → Update Video Thumbnail
   - Video ID: output from step 2
   - Thumbnail URL: `{{thumbnail_url}}`

**Paste Hook URL into n8n node:** `Zapier → YouTube Upload` → URL field
