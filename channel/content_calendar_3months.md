## DataTable Schema (n8n)
```
اسم الجدول: content_calendar

الحقول:
- id (auto)
- title (string) — عنوان الفيديو
- description (string) — الوصف (500-1000 حرف)
- tags (string) — وسوم مفصولة بفاصلة
- category (string) — money/investment/crypto/marketing/cars/partnership
- youtube_category_id (string) — 22=People, 25=News, 27=Education, 28=Science
- video_file_url (string) — رابط ملف الفيديو (Google Drive/S3)
- thumbnail_url (string) — رابط الصورة المصغرة
- publish_date (string) — YYYY-MM-DD
- status (string) — pending/uploading/uploaded/failed
- uploaded_at (string) — وقت الرفع الفعلي
- week (number) — رقم الأسبوع (1-13)
- video_number (number) — رقم الفيديو (1-52)
```

## إعداد Zapier Zap (خطوات)
1. **Trigger:** Webhooks by Zapier → Catch Hook
2. **Action:** YouTube → Upload Video
   - Title: `{{title}}`
   - Description: `{{description}}`
   - Tags: `{{tags}}`
   - Category: `{{category_id}}`
   - Video File: `{{video_file_url}}`
   - Privacy: Public
3. **Action:** YouTube → Update Video Thumbnail
   - Video ID: من الخطوة السابقة
   - Thumbnail: `{{thumbnail_url}}`

**انسخ Hook URL وضعه في n8n في خانة:**
`Zapier → رفع YouTube` → URL
