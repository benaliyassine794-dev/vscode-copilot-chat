# Money Mind – Thumbnail Design Templates & Guidelines

## Brand Color System
| Color | Hex | Usage |
|-------|-----|-------|
| Navy Blue | `#1A237E` | Background (primary), text overlay backgrounds |
| Gold | `#FFD700` | Primary text, accents, dollar signs |
| White | `#FFFFFF` | Secondary text, sub-headlines |
| Red | `#D32F2F` | "WRONG" / "MYTH" stamps, warnings |
| Green | `#2E7D32` | "CORRECT" / positive outcomes, profit numbers |
| Light Blue | `#E3F2FD` | Background variant (brighter mood) |

## Font System
- **Primary headline:** Montserrat ExtraBold or Anton (free on Google Fonts)
- **Sub-headline:** Montserrat Bold
- **Numbers/stats:** Bebas Neue (tall, impactful)
- **Min size on thumbnail:** 60px (must be legible on mobile at 168×94px preview)

## Template 1 — Comparison (A vs B)

**Used for:** Renting vs Buying, Buy vs Lease, Freelance vs 9-to-5, 800 vs 600 Credit Score

```
┌─────────────────────────────────────────────────┐
│   [LEFT HALF - Navy]     │   [RIGHT HALF - Gold] │
│                          │                       │
│   Character A            │   Character B         │
│   (stressed/losing)      │   (confident/winning) │
│                          │                       │
│   "OPTION A"             │   "OPTION B"          │
│   [in red/grey]          │   [in gold/green]     │
│                          │                       │
│         VS               │                       │
│   [white, center]        │                       │
│                          │                       │
│   SUB-TEXT: "The $X Difference" — centered, gold │
└─────────────────────────────────────────────────┘
```

**Character poses:**
- Left (losing side): slumped, confused, looking at bills, frowning
- Right (winning side): standing tall, smiling, holding keys/trophy/chart going up

**Example: Renting vs Buying**
- Left: character paying rent, money flying away, red background tint
- Right: character in front of house, gold background tint
- Center overlay: "VS" in white
- Bottom: "The $400,000 Difference"

## Template 2 — Main Educational (Big Number/Promise)

**Used for:** $1M by 40, How to Invest $100, Salary Negotiation

```
┌─────────────────────────────────────────────────┐
│   [Full navy background]                        │
│                                                 │
│   CHARACTER: Right side, confident pose         │
│   (pointing up, looking at text, arms open)     │
│                                                 │
│   HEADLINE: Left side, massive text             │
│   "$1,000,000"  ← Bebas Neue, gold, giant       │
│   "BY AGE 40"   ← Montserrat Bold, white        │
│                                                 │
│   SUB-LINE: "The Exact Plan"                    │
│             Montserrat, gold, smaller           │
│                                                 │
│   [LOGO: "Money Mind" bottom left, small]       │
└─────────────────────────────────────────────────┘
```

**Visual hierarchy:**
1. Large number (eye goes here first)
2. Character face (confirms emotion)
3. Supporting text (explains what the number means)

**Character variations:**
- Pointing at the number
- Holding a chart/graph showing upward trend
- Sitting at laptop with money graphics
- Standing confidently with arms folded

## Character Design Specifications

### The Money Mind Character
- **Style:** 2D flat animation, Clean Minimal
- **Gender:** Neutral/ambiguous — works for all audiences
- **Skin:** Multiple variants (3-4 skin tones available)
- **Clothing:** Business casual — collared shirt, neat appearance
- **Hair:** Simple, stylized, non-distracting

### Expression Library (needed for thumbnails)
| Expression | Used When |
|-----------|-----------|
| Confident Smile | Good outcome, comparison winner, achievement |
| Shocked/Surprised | Myth-bust, unexpected number reveal |
| Thinking (hand to chin) | Educational / "did you know" |
| Stressed (hands on head) | The "before" state, the problem |
| Celebratory (fist pump) | Goal achieved, quick tip success |
| Pointing Up | Directing attention to a number/chart |
| Pointing at Camera | Direct engagement, "you should know this" |

### Props to always have ready
- Money/bills floating
- Upward trending chart/graph
- House silhouette
- Car silhouette
- Clock / calendar
- Piggy bank
- Laptop/phone showing portfolio
- Graduation cap (career content)

## Thumbnail Production Workflow

1. **Script → Hook extraction** — Identify the single most compelling visual claim from the script hook
2. **Template selection** — Match format (comparison/educational/myth/tip)
3. **Character selection** — Choose expression and pose
4. **Text finalization** — Max 5 words in headline, max 3 words in sub-text
5. **Color check** — Background, text, and accent must use brand palette
6. **Mobile test** — Zoom out to 25% — can you still read it?
7. **Upload to YouTube Studio** — As custom thumbnail before publishing

---

## A/B Testing Guide

After 48 hours, if CTR is below 4%:
1. Test a different headline (same visual)
2. Test character on opposite side
3. Change background from navy to gold (inverted)
4. Add a larger number if one exists in the video
5. Add "SHOCKING" or "TRUTH" text if theme supports it

Winning formula for finance YouTube thumbnails:
- **Dollar amount** + **shocked/confident face** = highest CTR combination in the niche
- Never use a thumbnail without a visible human face (animated or real)
- Red elements (warnings, X marks) increase clicks on myth-bust content

---

## Batch Production Schedule

Produce thumbnails in batches of 4 (one week's content) every Thursday:
- Thursday morning: finalize all 4 scripts for next week
- Thursday afternoon: produce 4 thumbnails
- Friday morning: quality check all 4 thumbnails on mobile
- Schedule uploads with pre-made thumbnails for Tue/Wed/Fri/Sun

This keeps a 1-week buffer — never scrambling for a thumbnail on upload day.
