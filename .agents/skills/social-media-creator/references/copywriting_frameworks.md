# Copywriting Frameworks for Social Media

## 1. AIDA — Attention · Interest · Desire · Action
Best for: YouTube descriptions, Facebook, LinkedIn, Instagram captions
```
[ATTENTION] Stop-the-scroll hook: question, bold claim, shocking stat
[INTEREST]  Expand the hook — show you understand the reader's problem
[DESIRE]    Paint the transformation / outcome they want
[ACTION]    Single, clear CTA
```
**Example:** "Most creators waste 10 hours/week on content. [A] They film, edit,
post — and wonder why nobody watches. [I] The top 1% batch everything in one day. [D]
Drop 'BATCH' in comments and I'll send you my system. [A]"

## 2. PAS — Problem · Agitate · Solution
Best for: TikTok scripts, X threads, YouTube hooks, Reels captions
```
[PROBLEM]  Name the exact pain point
[AGITATE]  Make the problem feel urgent / costly
[SOLUTION] Present your content/product as the answer
```

## 3. Hook-Story-CTA (HSC) — Short-Form Native
Best for: TikTok, Reels, Shorts, Stories
```
[HOOK]  First 1–3s: pattern interrupt + curiosity gap
[STORY] 10–80% of content: deliver the promised value
[CTA]   Last 5–10%: follow / comment / save / link in bio
```
**Hook formulas:**
- "I [did X] so you don't have to..."
- "[N] things I wish I knew before [topic]..."
- "Stop [common mistake]. Do this instead."
- "POV: You just [dream scenario]"
- "[Tool/Hack] is changing [niche]. Here's how."

## 4. BAB — Before · After · Bridge
Best for: transformation content, tutorials, case studies
```
[BEFORE] Where they are now (the struggle)
[AFTER]  Where they could be (the outcome)
[BRIDGE] Your method / content as the path between the two
```

## 5. PPPP — Picture · Promise · Prove · Push
Best for: LinkedIn, Facebook long-form, email repurposed as posts
```
[PICTURE] Vivid scenario the reader recognizes
[PROMISE] What you'll deliver
[PROVE]   Data, case study, screenshot as evidence
[PUSH]    Specific next step
```

## Emotional Triggers That Drive Engagement
- **Curiosity gap:** "The one thing nobody tells you about X..."
- **FOMO:** "Everyone in [niche] is doing this but you"
- **Relatability:** "When you [specific frustration]..."
- **Social proof:** "[Number] people already did this"
- **Controversy:** "Unpopular opinion: [counterintuitive statement]"
- **Transformation:** "I went from X to Y in Z days"
