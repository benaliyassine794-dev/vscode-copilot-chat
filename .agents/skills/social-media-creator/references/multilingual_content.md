## RTL Languages — Special Handling

Right-to-left languages require special treatment in text overlays and captions:
**Arabic · Hebrew · Farsi/Persian · Urdu · Pashto · Sindhi · Uyghur**

### In Video Editing
- Text overlay alignment: RIGHT align, not left
- Subtitles: enable RTL mode in CapCut / Premiere / DaVinci
- Animated text: ensure animation flows right-to-left
- Numbers remain LTR even inside RTL text (Arabic numerals)

### In Captions
- Start caption with RTL unicode marker: `‫` (RIGHT-TO-LEFT EMBEDDING)
- Or use `‏` HTML entity before RTL text
- Most platforms (TikTok, Instagram) auto-detect language direction

### CapCut RTL Setup
1. Add text → select language → choose Arabic/Hebrew/Urdu
2. Alignment auto-switches to RTL
3. Use "Basic" font that supports Arabic/Urdu script (avoid Latin-only fonts)

## Localization vs Translation

**Translation:** converting words from one language to another
**Localization:** adapting the ENTIRE message for the culture (idioms, humor, references, visuals)

### Localization Checklist
- [ ] Replace idioms with locally understood equivalents
- [ ] Check cultural sensitivity (colors, gestures, symbols, religion)
- [ ] Adapt humor — jokes rarely translate directly
- [ ] Use local currency, measurements, date formats
- [ ] Verify that hashtags make sense in target language
- [ ] Check if platform is popular in target market at all

### Cultural Sensitivity Quick-Reference
| Symbol/Gesture | Western Meaning | Alternative Meaning              |
|----------------|-----------------|----------------------------------|
| Thumbs up 👍   | Good / OK       | Offensive in parts of Middle East|
| OK sign 👌     | OK              | Offensive in Brazil, Turkey      |
| White color    | Purity / Clean  | Mourning in China, Korea, India  |
| Number 4       | Neutral         | Unlucky (death) in Japan/China   |
| Winking 😉     | Playful         | Flirtatious/disrespectful in some cultures |

## Multilingual Hashtag Strategy

- Use hashtags IN the target language, not just English translations
- Example for Arabic fitness content: `#رياضة` `#تمارين` `#لياقة_بدنية` alongside `#fitness` `#workout`
- Mix: 2 local-language niche tags + 2 English global tags
- Research: search native-language hashtags on the platform to verify activity

## Platform-Specific Language Notes

- **YouTube:** add translated title + description in Video Details → Subtitles
- **TikTok:** auto-translation captions (enable in settings) — review for accuracy
- **Instagram:** content language tag in Advanced Settings helps discovery
- **LinkedIn:** can publish in multiple languages using the translation feature
- **X/Twitter:** native language is auto-detected; no extra setup needed
- **Pinterest:** create separate boards per language/market

## Multilingual Content Calendar

For creators targeting multiple markets:
1. Produce in primary language (e.g., English)
2. Auto-translate captions/script (DeepL or ChatGPT)
3. Native speaker review (Preply, iTalki, or local team)
4. Add native-language on-screen text overlay
5. Schedule for peak hours IN the target timezone
6. Use translated hashtags + geotag target country
