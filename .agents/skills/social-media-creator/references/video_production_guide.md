# Video Production Guide

## Short-Form Template (TikTok / Reels / Shorts — 15–90s)
```
[0–3s]   HOOK    — Visual + verbal pattern interrupt
[3–10s]  SETUP   — Context/problem (why this matters)
[10–75s] PAYOFF  — Deliver the value (steps, reveal, story)
[75–90s] CTA     — Follow / comment keyword / link in bio
```

### Hook Types by Goal
| Goal       | Hook Type          | Example                                        |
|------------|--------------------|------------------------------------------------|
| Views      | Curiosity gap      | "Wait until you see what happens at the end..."|
| Comments   | Debate starter     | "Hot take: [controversial opinion]"            |
| Saves      | Cheat-sheet        | "Save this — 5 things to do before you post"  |
| Shares     | Relatable scenario | "POV: You're a creator with 0 views"          |
| Followers  | Value promise      | "Follow for daily [niche] tips that work"     |

## Long-Form Template (YouTube — 8–20 min)
```
[0:00–0:30]  COLD OPEN   — Best moment OR bold claim (no intro yet)
[0:30–1:30]  HOOK EXPAND — Why this topic matters RIGHT NOW
[1:30–2:00]  ROAD MAP    — "In this video I'll show you X, Y, Z"
[2:00–X:XX]  CONTENT     — Chapters with clear keyword-rich titles
[X:XX–end]   CTA STACK   — Like → Comment → Subscribe → Next video
```

## CTA Library
**Soft CTAs (mid-video):**
- "Comment below: do you do X or Y?"
- "Save this video — you'll want to come back to it."
- "Drop a ✅ if this is making sense."

**Hard CTAs (end):**
- "If this was useful, the subscribe button is right there — it's free."
- "Watch this video next — it goes deeper on [subtopic]."
- "I made a free [resource] — link is in the description."

## Thumbnail Concept Briefs
**Curiosity Thumbnail:** blurred result + shocked face + text overlay ("I Can't Believe This Worked")
**Transformation Thumbnail:** before (desaturated) + after (vivid) + arrow/VS divider
**Authority Thumbnail:** clean brand-color bg + face + title text + logo corner

**Thumbnail Rules:**
- Readable at 120px wide (mobile YouTube grid)
- Max 3 elements: face + text + background
- Bold sans-serif font, 80pt+ equivalent
- High contrast, single dominant color

## Video SEO Checklist
- [ ] Target keyword in title (first 60 chars)
- [ ] Keyword in first 100 chars of description
- [ ] File name: keyword-slug.mp4 before upload
- [ ] Tags: 5–8 specific + 3 broad
- [ ] End screen added at 20s from end
- [ ] Cards added at 20% and 70% of video
- [ ] Chapters/timestamps in description
- [ ] Pinned comment with CTA or resource link
