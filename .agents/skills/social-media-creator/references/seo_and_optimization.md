# SEO & Content Optimization

## YouTube Title Formulas (High CTR)
1. `"[N] Proven Ways to [Goal] (Without [Common Barrier])"`
2. `"How to [Achieve Goal] in [Timeframe] (Step by Step)"`
3. `"[Keyword] Tutorial for [Audience] [Year]"`
4. `"Why [Controversial Claim] (Most People Are Wrong)"`
5. `"I [Did Unusual Thing] for [N Days] — Here's What Happened"`

**Rules:** 60 chars max (search truncates after); keyword in first 40 chars; Title Case; numbers beat spelled-out quantities.

## YouTube Description Template

## Keyword Research Workflow
1. **Seed keyword** — main topic
2. **Platform autocomplete** — type keyword, note all suggestions
3. **TubeBuddy / vidIQ** — competition score and search volume
4. **Google Trends** — compare variants, find rising terms
5. **Answer The Public** — question-based keywords for titles
6. **Competitor analysis** — top 10 videos in niche → extract title patterns

## Instagram / TikTok SEO (Text Indexing)
Both platforms now index captions, on-screen text, and spoken words:
- Use target keyword in caption first line
- Add keyword as on-screen text overlay in first 3 seconds
- Speak keyword clearly in first 15 seconds (auto-captions are indexed)
- Instagram: add alt text with keyword (Edit → Advanced → Alt Text)
